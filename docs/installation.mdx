## CLI

<Tabs>
  <TabItem label="npx (no install)">

```bash
npx @cerberauth/stubidp --redirect-uri http://localhost:8080/callback
```

  </TabItem>
  <TabItem label="npm global">

```bash
npm install -g @cerberauth/stubidp
```

Then run:

```bash
stubidp --redirect-uri http://localhost:8080/callback
```

  </TabItem>
  <TabItem label="Dev dependency">

```bash
npm install --save-dev @cerberauth/stubidp
```

Add a script to your `package.json`:

```json
{
  "scripts": {
    "idp": "stubidp --redirect-uri http://localhost:8080/callback"
  }
}
```

  </TabItem>
</Tabs>

Verify the server is running:

```bash
curl http://localhost:3000/oauth2/.well-known/openid-configuration
```

<Aside type="caution">
  stubIdP is intended **for development and testing only**. Do not use it as a production identity provider.
</Aside>
