## Quick start

```bash
npx @cerberauth/stubidp \
  --client-id web-app \
  --client-secret web-app-secret \
  --redirect-uri http://localhost:8080/callback
```

Your OIDC provider is live at `http://localhost:3000/oauth2`.

OIDC discovery: `http://localhost:3000/oauth2/.well-known/openid-configuration`
